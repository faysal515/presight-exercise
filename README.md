# Presight exercise

Monorepo with a **Node/Express** API (`server`) and a **React + Vite** client (`client`).

## Requirements

- **Node.js** 20+ (LTS recommended)
- **Yarn** classic (v1), e.g. `npm install -g yarn`

## Run everything (2 commands)

From the repository root:

```bash
yarn
yarn dev
```

Fyi - mongodb memory server is being used for this project. first time script will download a mongodb binary. wait 5-6 seconds to complete.
